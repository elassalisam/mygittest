# Git Deployment Test Website

This is a simple test website designed to verify DirectAdmin Git deployment functionality.

## Features

- ✨ Modern, responsive design
- 🎨 Gradient color scheme (Cyan to Purple)
- 📱 Mobile-friendly layout
- ⚡ Fast loading with minimal dependencies
- 🔄 Smooth animations and transitions

## Files

- `index.html` - Main HTML structure
- `style.css` - All styling and responsive design
- `script.js` - Interactive JavaScript features
- `README.md` - This file

## Deployment

This website is designed to be deployed via DirectAdmin's Git integration:

1. Create a GitHub repository
2. Push these files to the repository
3. In DirectAdmin, add the repository
4. Configure deploy branch and directory
5. Click Fetch, then Deploy
